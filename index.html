2021-2022 NHL GOALTENDER STATISTICS - ADVANCED STATS API

TOP NHL GOALIES

There is one commonality between the Stanley Cup Champions over the years, and that is elite goaltending. This API pulls all resources and advanced statistics for goaltenders from the 2021-22 NHL season, readily available and fully open for you to use.

This API pulls data for goaltenders who played a minimum of 40 games.

Please use: https://goalie-api-advanced-stats.herokuapp.com/api/goalie-name

WHAT YOU CAN EXPECT:
{"name":"Igor Shesterkin",
"gamesPlayed":53,
"goalsAgainst":106,
"expectedGoalsAgainst":140.1,
"GAA":2.07,
"puckFreezes":311}

NOTE: Calling the API endpoint without a goalie's FIRST and LAST name will return "unknown".
